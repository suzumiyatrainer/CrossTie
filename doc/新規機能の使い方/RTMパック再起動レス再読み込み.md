# RTMパック再起動レス再読み込み（動的リロード）機能

## 1. 概要
RTM（RealTrainMod）の追加車両パックやモデルパック、テクスチャをゲームに追加・更新・削除した際、マインクラフト本体を**再起動することなく、ゲーム内の操作のみで動的にパックデータを再読み込み（リロード）する機能**です。

従来は新しいモデルパックを導入するたびにゲームの起動に数分かけて再起動する必要がありましたが、本機能により数秒から数十秒の待機で最新のモデルが反映されるようになります。

---

## 2. 推奨される前提条件
- **実行場所**: **メインメニュー（ワールドに入っていない状態）**
  
> [!WARNING]
> ワールドに入った状態でリロードを実行しようとすると、パケット同期のズレやTileEntityの破損、グラフィックメモリのクラッシュを防ぐために**実行制限の警告**が表示されます。
> リロードを実行する際は、必ず一度ワールドを保存して終了し、メインメニュー画面に戻ってから実行してください。

---

## 3. 使用方法（手順）

1. **モデルパックファイルの配置**
   通常通り、マインクラフトの `mods` フォルダや `mods/modelpacks` 等の指定ディレクトリに、新しいRTMモデルパック（zipファイル）を追加、更新、あるいは削除します。

2. **Config画面を開く**
   マインクラフトのタイトル画面（メインメニュー）から、「Mods」一覧を開き、**CrossTie** を選択して「Config（設定）」画面を開きます。
   または、「設定」から**CrossTie** を選択すると、同様の画面に遷移できます。

3. **再読み込みの実行**
   設定画面の**「モデルパックの再読み込み (Reload Model Packs)」**ボタンをクリックします。
   - ワールドに接続していない状態であれば、確認の選択画面（はい / いいえ）が表示されます。
   - 「はい」をクリックすると、ゲームが専用の「再読み込み中...」待機画面（進捗ステータス表示）に切り替わります。

4. **処理の完了**
   待機画面のゲージまたはテキスト表示が完了し、設定画面に戻れば再読み込みは完了です。再度ワールドに入ると、新しく追加・変更した車両やモデルパックが即座に使用可能な状態になります。

---

## 4. 内部的な動作と安全性
本機能は、単にRTMのスレッドを再起動するだけでなく、以下のクリーンアップと高速化を内部で安全に行っています。
- **徹底的なキャッシュクリア**:
  `ModelPackManager` および `TextureManager` の持つモデルデータ・テクスチャマップ、また `KaizPatchX` が内部で保持するキャッシュ（`CachedPolygonModel` やLRUキャッシュ）を一度完全に破棄します。
- **ファイルロックの解除**:
  現在メモリ上で開かれている古い zip ファイルのファイルハンドルを安全にクローズし、Windows環境下でもファイルの移動や上書き更新がロックされずスムーズに行えるようにします。
- **ロード速度の最大化**:
  リロード処理の実行中のみ、一時的にRTMのモデルロード速度設定（`loadSpeed`）を最大値（3）に強制設定し、最速の並行マルチスレッドでモデル解析を完了させます（完了後に元の設定値へ自動復元）。
